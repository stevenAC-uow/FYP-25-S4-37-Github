Setup Instructions

    1.Create a folder on your desktop
    2.Open that folder in  VS Code , open terminal  run these commands step by step
    3.Clone the repo
      -  git clone https://github.com/stevenAC-uow/FYP-25-S4-37-Github.git
    4.Go to Project Folder
      - cd FYP-25-S4-37-Github
    5.Go to Backend Folder
        - cd FYP-25-S4-37-EduQuest/Backend
    6.Install Packages
         - npm install 

   7.create a .env file inside the Backend  folder and add 
    PORT=3000
    MONGO_URI= mongodb connection string

  8.Run backend server 
-- npm run dev
Output
    MongoDB connected
    Server running on port 3000
